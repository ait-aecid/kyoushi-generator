    Run Cyber Range Kyoushi Generator.

    Options:

    Name Type Description Default
    --help boolean Show this message and exit. False

    cr-kyoushi-generator apply¶

    Apply and generate the template.

    Usage:

    cr-kyoushi-generator apply [OPTIONS] SRC DEST

    Options:

    Name Type Description Default
    --model, -m directory The model directory for the TIM that is to be instantiated required
    --seed, -s integer Global seed for PRNGs used during context generation required
    --var text N/A required
    --var-file file N/A required
    --help boolean Show this message and exit. False
